# Terraform Hello World en AWS

Este proyecto despliega una aplicación "Hola Mundo" en AWS usando:

- **Terraform** para infraestructura como código
- **Docker** para contenerización
- **AWS EC2** para servidores virtuales
- **GitHub Actions** para CI/CD

## Estructura del Proyecto
TerraformHelloWorld/
├── .github/workflows/ # Automatización CI/CD
├── main.tf # Configuración principal
├── variables.tf # Variables
├── outputs.tf # Salidas
├── user_data.sh # Script de inicio
└── README.md # Este archivo

text

## Primeros Pasos

1. Clona este repositorio
2. Configura AWS CLI
3. Ejecuta `terraform init`
4. Ejecuta `terraform apply`
